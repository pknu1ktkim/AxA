⚙️

AI 엔진 조립하기

챗GPT의 심장인 '트랜스포머'를 라이브러리 없이 직접 코딩하며 AI의 작동 원리를 밑바닥부터 파헤칩니다.

⚖️

AI 다이어트 시키기

거대한 AI 모델을 성능은 유지하면서 크기만 줄여, 아주 가볍고 빠르게 작동하도록 최적화(경량화)합니다.

🏭

AI 자동화 공장 세우기

AI가 늘 최상의 실력을 유지하게 자동으로 재학습시키고 배포하는 MLOps 공정을 구축합니다.

훈련 목표

Transformer 모델을 라이브러리 최소화 기반으로 직접 구현(Scratch Build)하고, PEFT 기반 모델 최적화 역량을 확보합니다.

단순 모델링을 넘어 데이터 파이프라인부터 MLOps 인프라(Triton, MLflow)까지 전 과정을 직접 설계하고 구현하는 실전 풀스택 코어 인재를 육성하는 것이 본 과정의 핵심입니다.

🔥 핵심 역량 (Core Tech)

  • • Transformer 구조 수학적 원리 분석
  • • Triton Inference Server 고성능 서빙
  • • LoRA/QLoRA 모델 경량화 전략
  • • GitHub Actions 기반 MLOps CI/CD
